# Tinderazo
Aplicación de citas para gente que no es superficial
## Comenzando 🚀️
Instrucciones para obtener una copia del proyecto en funcionamiento en tu máquina local para propósitos de desarrollo y pruebas.
### Prerrequisitos 📋️
    $ npm install

To run the app:

    $ npm run dev
